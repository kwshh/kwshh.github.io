# 拆机记录–笔记本加固态 r9000p

简单记录过程和以及可能需要的注意的地方

# 工具：

细小螺丝刀一把，（可能是 1mm 粗细，自己最细的螺丝刀反正是不行的）

稍微硬一点的塑料片，也可以用身份证银行卡材质之类的卡片，目的是可以穿过缝隙，硬度又支持翘起来

# 过程：

# 1，拆后盖

建议先从两侧出口风出发，到后出风口，最后前侧的盖板；因为前面翘的时候可以立起来，最后一侧再放平翘，翘完不会让板子突然落下来。

# 2，装固态

需要注意的是，放手上静电（触摸金属即可），断电源线，再开机 2、3 次（彻底释放静电），然后就可以装固态啦！最后装完，插上电池，先开机看看此电脑的可分配硬盘空间，然后再盖上盖子就 OK 啦！

# 结束：

开机会发现有些功能会重置，这里我比较明显的地方是屏幕颜色明显不对了，然后打开个性化进入 HDR 设置就突然恢复了。应该没有什么大问题，以上就是个人对 R9000P 拆机安装过程的一点记录。
